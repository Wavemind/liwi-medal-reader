Commandes
=======

Install
```
npm i
```

Launch React Native bundler
```
react-native run-ios
```

Launch React Native app
```
react-native start run-ios --reset-cache
```

Generator documentation.md for React Component
```
npm run doc
```

launch documentation
``` 
cd documentarus
npm run start
```

E2E test
```
npm run detox-android
npm run detox-ios
```
